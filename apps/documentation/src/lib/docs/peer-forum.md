# Peer Discussion Forum

Aspiron's forum provides structured peer support for doubt resolution and knowledge sharing.

## Forum Structure

### Organization

Forum is organized by:

- **Exam** - PGT, JEE, NEET
- **Subject** - Physics, Chemistry, etc.
- **Topic** - Specific learning topic

### Thread Types

1. **Doubt Threads** - Ask questions
2. **Discussion Threads** - Topic discussions
3. **Resource Threads** - Share helpful content
4. **Announcements** - Official updates

## Posting

### Creating a Thread

1. Select exam, subject, topic
2. Choose thread type
3. Write title and description
4. Add images if needed
5. Attach notes or timestamps
6. Submit

### Attaching Content

Can attach:

- Images of doubts
- Video timestamps
- Notes excerpts
- Reference links

## Forum Features

### Upvoting

- Upvote helpful answers
- Sort by most helpful
- Recognize good contributors
- Quality content rises

### Marking Solutions

- Original poster marks answer
- Verified solution highlighted
- Helps future readers
- Rewards helpful peers

### Search

Search across forum:

- By keyword
- By topic
- By exam
- By unanswered

## Moderation

### Community Bot

Community Bot helps moderate:

- Duplicate detection
- Spam prevention
- Category suggestions
- Encourages quality

### Manual Moderation

> **Current**: Manual moderation in MVP

- Flag inappropriate content
- Remove spam
- Guide poster to improve
- Escalate serious issues

### Community Guidelines

- Be respectful
- Stay on topic
- Help others learn
- No spam or promotion
- Credit sources

## Forum and Context

### Context-Aware

Forum respects your context:

- Shows relevant threads only
- Posts in your exam/subject
- Relevant suggestions
- Filter by your needs

### Thread Visibility

- Visible to same exam/subject students
- Private to your context
- Not visible to others
- Privacy-first approach

## Benefits

### Doubt Resolution

- Get answers from peers
- Multiple perspectives
- Quick responses
- Community knowledge

### Knowledge Sharing

- Share what you know
- Help others learn
- Build reputation
- Collective intelligence

### No Social Noise

Unlike social media:

- No likes or followers
- No viral mechanics
- No distraction features
- Focus on learning
