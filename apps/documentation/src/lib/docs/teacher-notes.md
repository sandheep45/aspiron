# Teacher Notes & PDFs

Aspiron provides official teacher-created notes and PDF materials for structured learning.

## PDF Features

### PDF Viewer

Built-in viewer with:

- **Zoom In/Out** - Adjust magnification (pinch or buttons)
- **Continuous Scroll** - Smooth vertical navigation
- **Page Navigation** - Jump to specific page number
- **Thumbs View** - Visual page overview
- **Search** - Find text within PDF
- **Print** - Print selected pages (future)

### PDF Controls

- Sidebar toggle
- Fullscreen mode
- Dark mode support
- Horizontal/vertical scroll

## Teacher Notes

### Content Types

- **Chapter Summaries** - Overview of key concepts
- **Theory Notes** - Detailed explanations
- **Formula Sheets** - Important formulas
- **Important Questions** - Key practice questions
- **Revision Notes** - Quick review materials

### Organization

Notes are organized by:

- Subject
- Chapter
- Topic
- Note type

### Quality Standards

All teacher notes are:

- Syllabus-aligned
- Exam-focused
- Verified for accuracy
- Regularly updated

## Offline Access

### Download for Offline

- Download PDFs for offline viewing
- Available in downloads section
- Download progress indicator
- Storage management tools

### Offline Limitations

- Full offline mode (future)
- Currently: Viewer online, PDF cached
- Video content: Online only in MVP

## PDF Integration

### Linked to Content

Teacher PDFs are linked to:

- **Videos** - Related video lessons
- **Topics** - Associated learning topics
- **Quizzes** - Practice questions
- **Notes** - Student note sections

### Timestamps

Some PDFs include:

- Video timestamps
- Practice question references
- Related concept links

## Sharing and Access

### Availability

Teacher notes are:

- Free for all students
- Linked to context (exam/subject)
- Accessible from topic pages
- Searchable

### Restrictions

- Cannot be downloaded (MVP)
- Cannot be shared externally
- Cannot be printed (future)
