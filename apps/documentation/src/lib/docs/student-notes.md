# Student Notes

Aspiron allows students to create personal notes with timestamps for precise recall.

## Note Creation

### While Watching Videos

1. Click "Add Note" button
2. Note is created at current timestamp
3. Add your content
4. Save

Result: Note is linked to exact video position.

### While Reading PDFs

1. Open teacher PDF
2. Click "Add Note" on specific page
3. Add your content
4. Save

Result: Note linked to PDF page.

### While Solving Quizzes

1. During quiz review
2. Click "Add Note" on question
3. Add insight or reminder
4. Save

Result: Note linked to specific question.

## Note Features

### Timestamps

Each note has:

- **Video Timestamp** - Exact position in video
- **PDF Page** - Reference to PDF page
- **Question ID** - Linked quiz question
- **Created At** - When note was created
- **Updated At** - Last modified

### Navigation

- Click timestamp to jump to video position
- Click page number to open PDF at page
- Click question to open quiz
- One-click return to note

## Note Organization

### Auto-Organization

Notes are automatically organized by:

- **Exam** - Your selected exam
- **Subject** - Your chosen subject
- **Topic** - Associated learning topic
- **Source** - Video, PDF, or quiz

### Manual Organization

Students can:

- Add custom tags
- Create note collections
- Link related notes
- Rename notes

## Note Usage

### During Review

1. Open your notes
2. Click timestamp to return to video
3. Review the context
4. Edit note if needed

### While Learning

1. Watch video
2. Create note at important moment
3. Add your insights
4. Continue learning

### For Revision

1. Browse notes by topic
2. Click timestamps for context
3. Identify gaps
4. Add revision notes

## Note Sharing

### Private Notes

Default: Notes are private

- Only visible to creator
- Not accessible to anyone
- Fully personal

### Sharing Options

Students can:

- Share with specific peers
- Make public to community
- Attach to forum posts

## AI Integration

### AI Context

AI knows your notes:

- Can explain your notes back to you
- Can find related content
- Can suggest improvements
- Can summarize note content

### AI Features

AI uses notes for:

- Personalized explanations
- Recall Check questions
- Study recommendations
- Gap identification
