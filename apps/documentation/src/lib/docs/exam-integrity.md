# Exam Integrity & Safety

Aspiron ensures exam integrity through technical safeguards and ethical principles.

## Integrity Measures

### AI Disable During Exams

AI features are automatically disabled:

- No AI chat during exams
- No AI explanations during exams
- No AI suggestions during exams
- Pure student effort required

### Copy/Paste Blocking

During exams:

- Copy functionality disabled
- Paste functionality disabled
- Right-click menu restricted
- Keyboard shortcuts blocked

### External Tools

Prevention of:

- Screen recording
- External browsers
- Chat applications
- Answer sharing

## Technical Safeguards

### Exam Mode

When taking exams:

- Fullscreen required
- Timer enforced
- Navigation restricted
- AI disabled

### Proctoring Integration

Works with AI proctoring:

- Focus score generation
- Tab switch detection
- Suspicious behavior flags
- No real-time intervention

## Student Transparency

### Clear Communication

Students are informed:

- "AI will be disabled during exam"
- "Copy/paste is blocked"
- "Your focus will be monitored"
- "This is for your benefit"

### No Surprises

Transparency rules:

- Clear before exam starts
- Settings explained
- Purpose stated
- Questions answered

## Integrity Boundaries

### What Happens If...

| Situation | Response |
|-----------|----------|
| Tab switch | Logged, not penalized |
| Unusual pace | Flagged for self-awareness |
| Focus issues | Reported to student only |
| Suspicious patterns | Noted, no action in MVP |

### MVP Approach

Trust-first methodology:

- No exam cancellation
- No score reduction
- No reporting to others
- Only self-awareness tools

### Future Considerations

> **Planned**: May evolve based on feedback

- Peer comparison (anonymous)
- Achievement for integrity
- Trust scores
- Enhanced monitoring

## Data and Privacy

### What Is Monitored

During exams:

- Tab switching
- Time on questions
- Answer patterns
- Focus duration

### What Is NOT Monitored

Never collected:

- Screen content
- Camera or microphone
- Personal data
- External communications

### Data Usage

Monitored data used for:

- Focus score calculation
- Integrity indicators
- Personal improvement
- NOT shared or sold

## Student Rights

### During Exam

Students have the right to:

- Clear rules explained
- Technical issues resolved
- Fair assessment
- Privacy protection

### After Exam

Students can:

- View focus score
- See integrity report
- Understand flagged behavior
- Improve for next time
