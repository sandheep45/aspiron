# Live Classes

Aspiron's live classes provide real-time interactive learning sessions with teachers.

## Live Class Features

### Class Structure

- **Scheduled Sessions** - Pre-announced class times
- **One-Way Video** - Teacher streams to students
- **Live Chat** - Real-time text communication
- **Duration** - Typically 60-90 minutes

### During Class

#### Video Stream

- High-quality video from teacher
- Screen sharing for explanations
- Document camera for problem-solving
- Adaptive quality based on connection

#### Live Chat

- Students can ask questions
- Teachers respond during or after class
- Moderated by Community Bot
- Organized by topic

### Class Recordings

All live classes are automatically recorded:

- Available immediately after class
- Same features as recorded videos
- Timestamped chapters
- AI-generated summaries

## AI Features

### AI Summaries

After each class, AI generates:

- **Class Summary** - Key topics covered
- **Key Points** - Important concepts explained
- **Missed-Class Recap** - What was missed if absent
- **Action Items** - Suggested follow-up activities

### AI Chat Integration

Students can ask AI about the class:

- "What was covered in today's class?"
- "Explain the third concept again"
- "What questions were asked in chat?"

## Live Class Access

### Schedule

Classes are announced in advance:

- Weekly schedule published
- Calendar integration
- Reminder notifications
- Multiple time zones supported

### Joining Class

- Click join button at scheduled time
- Check audio/video before joining
- Join from any device
- Stable internet required

## Chat Features

### During Class Chat

- Text-based communication
- Emoji reactions
- Question pinning
- Moderation tools

### Chat Moderation

Community Bot helps moderate:

- Flags inappropriate messages
- Removes spam
- Promotes unanswered questions
- Summarizes common questions

## Class Interaction

### Student Participation

- Ask questions via chat
- React to important points
- Take notes (auto-timestamped)
- Complete post-class quizzes

### Teacher Tools

- See participant count
- View question queue
- Pin important questions
- Share resources in chat

## Post-Class Features

### Recording Access

- Re-watch anytime
- Same interactive features as videos
- Bookmark important moments
- Share specific timestamps

### AI Summaries

- Automatic summary generation
- Key concepts extraction
- Timestamp-linked sections
- Study guide creation

### Follow-Up

- Quiz related to class content
- Practice problems
- Additional resources
- Next class preparation
