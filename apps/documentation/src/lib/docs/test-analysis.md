# Test Analysis

After each test or mock exam, AI provides detailed analysis to help you improve.

## Analysis Overview

### What Gets Analyzed

After test completion:

- Overall score
- Accuracy by topic
- Time management
- Mistake patterns
- Strengths and weaknesses

### Analysis Components

1. **Performance Summary**
2. **Topic Breakdown**
3. **Time Analysis**
4. **Mistake Patterns**
5. **Improvement Plan**

## Performance Summary

### Score Overview

You see:

- Overall percentage
- Total correct/incorrect
- Marks obtained vs. total
- Comparison to previous attempts

### Performance Indicators

- Strong areas highlighted
- Weak areas identified
- Improvement shown
- Consistency measured

## Topic Breakdown

### Accuracy by Topic

For each topic:

- Questions attempted
- Correct answers
- Accuracy percentage
- Trend (improving/declining)

### Priority Topics

AI identifies:

- Most problematic topics
- High-weight topics struggled
- Quick wins (easy topics missed)
- Focus areas

## Time Analysis

### Time Management

Shows:

- Total time taken
- Average time per question
- Time on wrong vs. correct
- Time pressure points

### Suggestions

AI suggests:

- Speed up on easy questions
- Spend more time on hard ones
- Practice time management
- Skip and return strategy

## Mistake Patterns

### Pattern Identification

AI finds:

- Common error types
- Conceptual mistakes
- Calculation errors
- Misreading questions

### Pattern Examples

- "Often mistakes in thermodynamics"
- "struggles with integration"
- "rushing on multiple choice"
- "misinterprets assertion questions"

## Improvement Plan

### "What To Do Next"

Clear actionable steps:

1. **Review Topics** - Specific topics to revisit
2. **Watch Videos** - Specific videos (with links)
3. **Practice Questions** - Targeted quizzes
4. **Revision Focus** - What to review
5. **Next Test** - When to attempt next

### AI-Generated Plan

Personalized based on:

- Performance data
- Weak areas
- Available time
- Target exam date

## Historical Analysis

### Progress Over Time

Track improvement:

- Accuracy trends
- Score progression
- Topic mastery
- Time management

### Benchmark Comparison

Compare to:

- Previous self-performance
- Anonymous peer average (future)
- Target score needed
