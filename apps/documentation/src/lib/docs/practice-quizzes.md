# Practice Quizzes

Aspiron's practice quizzes reinforce learning through active recall and immediate feedback.

## Quiz Types

### Multiple Choice Questions (MCQs)

- Single correct answer
- Multiple correct options (select all)
- Assertion-reasoning
- Match the following (future)

### Numerical Questions

- Integer answers
- Decimal answers
- Significant figures
- Step-by-step solutions

### Subject-Specific

- Physics: Numerical problems
- Chemistry: Equation balancing
- Mathematics: Proof-based questions
- Biology: Diagram-based (future)

## Quiz Features

### Timed Practice

- Topic-wise timers
- Section-wise timers
- Full practice mode
- Untimed practice option

### Question Navigation

- Question palette
- Mark for review
- Skip and return
- Time remaining indicator

### Immediate Feedback

After submission:

- Correct/incorrect indicator
- Your answer vs. correct answer
- Time taken per question
- Difficulty level

## Solutions

### Step-by-Step Solutions

After submission, students see:

- **Explanation** - Why answer is correct
- **Step-by-Step** - Detailed solution process
- **Concept Link** - Related video/topic
- **Similar Questions** - Practice more

### AI Explanations

> **Current**: AI explanations available after submission

AI provides:

- Alternative explanations
- Concept breakdowns
- Common mistakes
- Related formulas

## Quiz Organization

### By Topic

Quizzes organized by:

- Learning topics
- Chapter sections
- Difficulty levels
- Question types

### By Difficulty

- **Easy** - Basic concept testing
- **Medium** - Application questions
- **Hard** - Advanced problem-solving
- **Mixed** - All difficulty levels

## Quiz Analytics

### Performance Metrics

After each quiz:

- **Score** - Percentage correct
- **Accuracy** - Overall performance
- **Time Analysis** - Time per question
- **Topic Breakdown** - Performance by topic

### Improvement Tracking

- Quiz history
- Accuracy trends
- Weak areas identified
- Strengths highlighted

## Quiz and Context

### Exam-Specific

Quizzes match your exam:

- Question patterns
- Marking schemes
- Time limits
- Difficulty level

### Subject-Relevant

Quizzes adapt to:

- Subject requirements
- Syllabus coverage
- Previous performance
- Learning progress

## Quiz Tips

### Before Quiz

- Review related videos
- Read teacher notes
- Understand concepts
- Don't guess

### During Quiz

- Read questions carefully
- Manage time wisely
- Mark and return
- Review marked questions

### After Quiz

- Review solutions
- Understand mistakes
- Learn from errors
- Practice similar questions
