# Tests & Mock Exams

Aspiron provides comprehensive assessment through chapter tests and full-length mock exams.

## Test Types

### Chapter Tests

- Cover single chapter
- 20-50 questions
- 30-60 minutes
- Topic-specific

### Sectional Tests

- Cover multiple chapters
- 30-60 questions
- 60-90 minutes
- Section-specific

### Full Mock Exams

- Complete exam simulation
- Full question count
- Real exam duration
- All subjects

## Exam Features

### Real Exam UI

- Similar interface to actual exam
- Navigation palette
- Question numbers
- Time display
- Submit button

### Exam Controls

- Start exam
- Pause (limited)
- Submit
- Exit

### During Exam

- Cannot go back (optional)
- Timer countdown
- Question progress
- Review marked questions

## Exam Integrity

### AI Proctoring

During mock exams:

- **Tab Switching Monitoring** - Tracks active tab
- **Answer Speed Detection** - Flags unusual patterns
- **Pause Detection** - Monitors extended pauses
- **Focus Score** - Overall attention metric

### Integrity Features

- AI disabled during exams
- Copy/paste blocked
- No external tools
- Transparent monitoring

### Integrity Output

After exam:

- **Focus Score** - Attention metric
- **Integrity Indicators** - Any concerns
- **No Punishment** - Trust-first approach
- **Personal Report** - For self-improvement

## Post-Exam Analysis

### Score Report

- Overall score
- Section-wise breakdown
- Topic-wise performance
- Accuracy percentage

### Time Analysis

- Total time taken
- Time per question
- Time management insights
- Suggestions for improvement

### Mistake Patterns

- Common error types
- Topics with most mistakes
- Conceptual gaps
- Pattern identification

### AI Recommendations

AI provides:

- What to study next
- Topics to revise
- Practice questions
- Video recommendations

## Exam Modes

### Practice Mode

- No time limit
- Review during exam
- Solutions visible
- AI assistance available

### Exam Mode

- Timed
- No review during
- Solutions after submission
- AI disabled

### Both Modes Available

Students choose based on:
- Learning vs. assessment goal
- Practice needs
- Comfort level
