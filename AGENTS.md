# Aspiron ‚Äî Agent Guide

## Workspace

- **pnpm** monorepo (`pnpm@10.28.0`), hoisted node linker (`nodeLinker: hoisted`)
- JS workspace: `apps/*` (web-admin, mobile-student, documentation) and `packages/*` (config, api-client, tanstack-client)
- Rust workspace: `apps/backend` (Axum API server), `apps/migrations` (SeaORM migrations)
- **Biome** for JS linting/formatting (no ESLint/Prettier). Config at `biome.json`.
- **Just** (`just`) is the task runner ‚Äî runs before every commit via `husky` + `lint-staged` (`just ci`)

## Commands

### JS
| Command | What |
|---|---|
| `pnpm dev` | Run all JS apps in parallel (via root `dev:all` script) |
| `pnpm build` | Build all JS packages + apps (via root `build:all`) |
| `pnpm test --workspaces` | Run JS tests across workspace |
| `pnpm biome check .` | Lint + format check (no mutations) |
| `pnpm biome lint --write --unsafe .` | Auto-fix + format |

### Rust
| Command | What |
|---|---|
| `cargo build` | Build all Rust crates |
| `cargo test -p backend` | Run backend tests |
| `just seed` | Seed database (requires `seed` feature) |
| `just migrate <args>` | Run migrations (`just migrate -- reset` to reset, `just migrate` to apply) |
| `just fresh-db` | Reset + migrate + seed in one shot |

### Just recipes
| Recipe | What |
|---|---|
| `just ci` | format ‚Üí lint ‚Üí build-all ‚Üí check (runs on commit) |
| `just format` | `cargo fmt` + `biome lint --write --unsafe` |
| `just lint` | `cargo clippy -D warnings` + `biome lint` |
| `just check` | `cargo check` + `biome check` |
| `just build` | Parallel Rust + JS builds |
| `just test` | Parallel Rust + JS tests |
| `just setup` | Create `.env`, SSL certs, install deps, setup DB |
| `just generate-types` | Generate TS types from Rust DTOs via `ts-rs` (see below) |

## Architecture

### Monorepo boundaries
```
packages/config           ‚Üí @aspiron/config        ‚Äî env validation (Zod + @t3-oss)
packages/api-client       ‚Üí @aspiron/api-client    ‚Äî Axios HTTP client + generated types
packages/tanstack-client  ‚Üí @aspiron/tanstack-client ‚Äî TanStack Query hooks wrapping api-client
apps/web-admin            ‚Äî React 19 + TanStack Start (SSR) ‚Äî main admin dashboard
apps/mobile-student       ‚Äî Expo / React Native ‚Äî student app (mostly template code)
apps/backend              ‚Äî Rust Axum server ‚Äî backend API
apps/migrations           ‚Äî SeaORM migrations
apps/documentation        ‚Äî SvelteKit docs site
```

### Frontend (web-admin) key facts
- **TanStack React Router** (file-based, generates `routeTree.gen.ts` ‚Äî do not edit manually)
- **Tailwind CSS v4** (no `tailwind.config.js` ‚Äî uses CSS-based config)
- `@/*` maps to `apps/web-admin/src/*`
- All private routes live under `src/routes/_private-routes/` and require auth
- Dashboard is at `src/routes/_private-routes/dashboard/index.tsx`
- Auth via Auth.js JWT with credentials provider (Auth.js catch-all at `src/routes/api/auth/$`)
- SSR enabled via `@tanstack/react-start` with Vite (rolldown-vite)
- React Compiler enabled via `babel-plugin-react-compiler`

### API Client + TanStack Query patterns
- New API service: add file to `packages/api-client/src/services/<domain>/<name>.service.ts` following existing patterns
- New Query hook: add file to `packages/tanstack-client/src/hooks/<domain>/<name>.ts` using `useAxiosConfig()`, add query key to `query-keys.ts`
- Always rebuild packages after changes: `just build-packages` (config ‚Üí api-client ‚Üí tanstack-client)
- The `tanstack-client` build has a known UMD resolution issue; verify with `pnpm --filter @aspiron/tanstack-client build`

### Backend (Rust) key facts
- Axum 0.8, SeaORM 1.1, PostgreSQL 16+
- Service pattern: `handler.rs` ‚Üí `service.rs` ‚Üí `repository.rs` for each domain module
- `ts-rs` generates TypeScript types from Rust DTOs via `#[derive(TS)]` ‚Äî run `just generate-types` after DTO changes
- Clippy denies: `unwrap`, `expect`, `panic`, `todo`, `dbg!`, `print!`. Use proper error handling.
- All routes under `/api/v1/`

## Commit conventions
- Conventional commits enforced by `commitlint` (husky `commit-msg` hook)
- Pre-commit hook runs `just ci` via `lint-staged` ‚Äî this includes `format ‚Üí lint ‚Üí build-all ‚Üí check`
- Pre-commit build includes ALL Rust + JS packages; can be slow. `--no-verify` to bypass (only if hook failure is pre-existing).

## Database
- Docker services: PostgreSQL, Meilisearch, pgAdmin, LocalStack (see `docker-compose.yml`)
- Migrations via SeaORM in `apps/migrations`
- Seeding via `cargo run -p backend --features seed --bin seed -- <type>`
- `just fresh-db` = reset ‚Üí migrate ‚Üí seed

## Generated code (do not edit)
- `apps/web-admin/src/routeTree.gen.ts` ‚Äî auto-generated by TanStack Router
- `packages/api-client/src/generated-types/` ‚Äî auto-generated by `ts-rs` via `just generate-types`
- `apps/backend/src/seeds/generated_passwords.json` ‚Äî gitignored, checked by Biome (ignore formatting warnings)
